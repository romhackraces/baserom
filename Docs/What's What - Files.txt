-------------
Baserom Files
-------------

Main Folder
-----------

RHR4.bps - the baserom itself is contained in this patch
RHR4.m16 - internal regular sprite tiles for Lunar Magic
RHR4.s16 - internal custom sprite tiles for Lunar Magic
RHR4.dsc - custom block descriptions as seen in Lunar Magic (generated by GPS)

lunar-monitor-config.txt - configuration file for Lunar Monitor

!backup_hack.bat - build system script for backing up your hack
!build_hack.bat - a legacy script to build your hack
!initialize_baserom.bat - initializes the baserom by downloading all the tools
!restore_from_backups.bat - script to help restore items into your ROM from a backup

@build_script_common - a file to keep definitions and routines common to the build scripts


Other
-----

.gitignore - as the baserom is managed on GitHub, this file is a list of files to tell git to ignore
.gitkeep - files put in otherwise empty directories to keep it in git
